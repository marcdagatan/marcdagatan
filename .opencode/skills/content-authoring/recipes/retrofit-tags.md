# Recipe: Retrofit Tags (Existing Posts)

## When to use

- You added tag pages / related posts and need older posts to participate.

## Steps

1. Read the post and write a 1-sentence thesis.
2. Pick 2â€“5 tags that describe the _system_ topic, not the anecdote.
3. Normalize tags (lowercase, hyphens).
4. Update `description` to mention the constraint/failure mode.

## Minimal examples

Good tag sets:

- `["ai", "governance", "control-plane"]`
- `["hiring", "process", "failure-modes"]`

Bad tag sets:

- `["thoughts", "random", "life"]`
- `["ai", "gpt", "chatgpt", "llm"]` (too redundant)
